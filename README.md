# Catgirls Display

An Android app for browsing catgirl images inspired from NyarchLinux' Catgitls Downloader

## Features
- Random catgirl images from nekos.moe
- NSFW filtering
- Tag filtering
- Auto-next
- Save to gallery
- Star images for later (stored locally)
- Share images
- Fullscreen mode (tap for next, double-tap to exit)
- Swipe gestures for navigation

## Tech Stack cuz why not
- Kotlin
- Retrofit for API calls
- Glide for image loading
- Material Design components
- Coroutines for async operations

## API
Uses the [nekos.moe API](https://docs.nekos.moe/)
